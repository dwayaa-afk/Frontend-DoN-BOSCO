Introducción a CSS

Aprendiendo a dar estilo y diseño profesional a una página web

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que se utiliza para darle estilo a las páginas web. Mientras HTML organiza el contenido, CSS se encarga de la apariencia visual, como colores, tamaños, espacios y diseño.

¿Por qué utilizar CSS?

CSS permite mejorar la presentación de una página web, hacerla más atractiva y profesional. También facilita la organización del diseño y ayuda a que el sitio se adapte a diferentes dispositivos como computadoras y teléfonos.

Sintaxis básica

La sintaxis de CSS está formada por selectores y reglas. El selector indica qué elemento será modificado y las reglas definen los estilos que se aplicarán.

Ejemplo:

h1 {
  color: blue;
  font-size: 30px;
}

Formas de aplicar CSS

  • En línea: directamente en una etiqueta HTML.
  • Interno: dentro de la etiqueta style en el documento.
  • Externo: usando un archivo .css separado.

Propiedades esenciales de CSS

  • color → cambia el color del texto
  • background-color → cambia el fondo
  • font-size → tamaño del texto
  • text-align → alineación del texto
  • border-radius → bordes redondeados
  • display → organiza elementos
  • position → controla posiciones

Conclusión

CSS es una herramienta fundamental en el desarrollo web porque transforma páginas simples en sitios modernos, organizados y visualmente atractivos. Aprender sus bases es el primer paso para crear diseños más avanzados.
